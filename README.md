# Svelte-Esbuild-Electron template

A starter kit to build cross-platform desktop apps.

- Windows (portable executable)
- Linux (AppImage)

## Install
degit turtle/app
yarn install # or npm
cd electron
yarn install
```

## Developpement

run dev.bat
```


## Build for production

run build.bat

The desktop portable apps are built in the `dist/` directory.
